üìñ Understanding Parameters

Number of Simulations

How many game rounds to play for each game type. Each round drops 3 balls.

Example: 10,000 simulations = 10,000 rounds √ó 2 games = 20,000 total rounds
Card to Bet On

Choose which card (9, 10, J, Q, K, A) you're betting on for both games. Balls landing on this card count as "hits".

Both Fair and Tweaked games use the same chosen card
Bet Amount

Amount wagered per round (in dollars). Applied to both games. Payouts scale with bet size.

Total risk per round = bet √ó 2 games
Random Seed (Reproducibility)

Fixed starting point for random number generation. Both games use the same seed to ensure fair comparison. Same seed = same results every time you run. Uncheck for truly random outcomes each time.

Why this matters: Using the same seed ensures both games face identical "luck" conditions, making comparisons valid. Different seeds would introduce noise from random variation.
